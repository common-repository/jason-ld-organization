== Description ==

Jason LD Organization is a WordPress plugin that helps you to set parameters of
Organization schema using JSON-LD schema.org/Organization .
These are the fields you can set (if you need others, contact us)
Name (organization name) is the only required field
Telephone (organization telephone number)
Fax Number (organization fax number)
email (organization email)
With the Donation version you can also set the fields for Logo and Social
Profiles in Knowledge Graph

Test the final jason configuration at
https://developers.google.com/structured-data/testing-tool/

== Installation ==

1. Upload `jason-ld-org` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Organization information to the Jason LD Organization
